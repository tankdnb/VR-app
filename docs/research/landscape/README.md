# Landscape Research

This folder contains synthesis documents that sit above the individual
`wave-*` research passes.

Use these files when you want to understand:

- which repositories belong to the same `product family`;
- where `logical overlap` is high even if URLs are different;
- which repos are already covered well versus only partially;
- what should be studied next in a deeper pass.

## Files

- `vr-projects-master-index.md`  
  Broad combined index of tracked and historically added waves.

- `vr-projects-foundational-waves-1-7-retro-normalization.md`
  Canonical repeat deep-pass that normalizes the foundational `waves 1-7`
  corpus to the same standard used by the later GitHub waves.
