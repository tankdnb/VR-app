# Reality Window Experiment Track

This folder preserves the original `Reality Window` concept and the follow-up
engineering work around camera access, passthrough, and feasibility.

## Files

- `overlay-vr-mvp-spec.md`  
  Initial MVP specification for the camera-backed overlay window.

- `current-feasibility-status.md`  
  Rolling feasibility status across desktop `OpenVR`, `PICO/OpenXR`, and
  `Pico Connect` constraints.

- `pico-official-docs-direction.md`  
  Direction change triggered by the official PICO documentation.

- `pico-openxr-camera-path.md`  
  Findings from the official PICO OpenXR probing path.

- `pico-connect-constraint.md`  
  Why `Pico Connect` became a hard requirement for the original concept.

- `pico-connect-passthrough-spike.md`  
  Notes on passthrough coexistence testing during active streaming.
